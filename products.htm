Blue Swoop has built powerful Project Cost Reporting Software (Pluto) that integrates with MYOB and QuickBooks to automate cost reporting for any size project or production.

Pluto (desktop)                        Pluto Online coming soon...

Pluto is the production cost report management software application installed from CD or downloaded.

For each production there is a licensed Pluto data file.  This allows for multiple productions to be managed using Pluto and are fully transportable.

The locked off budget is entered in the Pluto data file.

All variance movements, narratives and period date ranges for rollover are entered and maintained in the Pluto data file.

Custom reports are set up in the accounting software (ie MYOB or QuickBooks), to allow the Chart of Accounts, Actual and Purchase Orders data to be uploaded from the accounting software to the Pluto data file.
